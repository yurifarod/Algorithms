Desenvolvimento e Parametrização do NSGA-II¶

De forma a seguir com o desenvolvimento e estudo das meta-heursticas foi proposto para esta atividade o desenvolvimento e teste de parmetros para uma das mais consolidadas tcnicas meta-heursticas aplicadas a problemas multiobjetivo, o Non-dominated sorting genetic algorithm II (NSGA-II).

Este algoritmo se trata de uma tcnica evolucionria e populocional baseada no algoritmo gentico utilizando o conceito de no dominncia de pareto. De forma similar ao,j desenvolvido, algoritmo gentico elitista, esta tcnica ordena as solues existentes na populao pelas caractertiscas da suas solues (aqui a dominncia de pareto), de forma que apenas uma parte da populao considerada de melhor qualidade permaence iterado, ou seja, perpetuando os seus genes e mutando.

Uma das grades referncias para o desenvolvimento deste trabalho foi o artigo encontrado no link:

https://www.iitk.ac.in/kangal/Deb_NSGA-II.pdf

def dtlz1_function(x, n_var, n_obj):
  f = []
  k = n_var - n_obj + 1

  g = 0
  i = n_var - k
  while (i < n_var):
    g = g + ((x[i] - 0.5) * (x[i] - 0.5) - (math.cos(20 * math.pi * (x[i] - 0.5))))
    i = i + 1
  
  g = 100 * (k + g)
  i = 0
  while (i < n_obj):
    v = (1.0 + g) * 0.5;
    f.append(v)
    i = i + 1

  i = 0
  j = 0
  while (i < n_obj):
    while (j < n_obj):
      f[i] = f[i] * x[j]
      j = j + 1
    if (i != 0):
      aux = n_obj - (i + 1)
      f[i] = f[i] * (1 - x[aux])
    i = i + 1

  return f

def dtlz2_function(x, n_var, n_obj):
  f = []
  k = n_var - n_obj + 1

  g = 0
  i = n_var - k
  while (i < n_var):
    g = g + ((x[i] - 0.5) * (x[i] - 0.5))
    i = i + 1
  
  i = 0
  while (i < n_obj):
    v = 1 + g;
    f.append(v)
    i = i + 1

  i = 0
  j = 0
  while (i < n_obj):
    while (j < n_obj):
      f[i] = f[i] * (math.cos(x[j] * 0.5 * math.pi));
      j = j + 1
    if (i != 0):
      aux = n_obj - (i + 1)
      f[i] = f[i] * (math.sin(x[aux] * 0.5 * math.pi))
    i = i + 1
  return f

Dominância de Pareto¶

Seja x1, x2 F, dizemos que x1 Pareto domina (ou, simplesmente, domina) x2 se f(x1) f(x2) e f(x1) 6= f(x2), ou seja, em pelo menos um dos objetivos a desigualdade atendida de forma estrita. Esta relao de dominncia escrita como f(x1) f(x2) (http://www.cpdee.ufmg.br/~lusoba/disciplinas/eee910/slides/lusoba/04-dominancia.pdf).

Podemos denotar por uma soluo eficiente ou pareto tima toda aquela soluo que no dominada por nenhuma outra. O conjunto de todas as solues no dominadas chamado de conjunto eficiente ou conjunto timo.

Para o desenvolvimento deste projeto foram desenvolvidas uma verso simplista da dominncia de pareto e a ordenao das solues segundo sua dominncia ou no.

#Algoritmo de Dominancia de Solues 
def dominancia_simples(sol_1, sol_2):
  domina = False
  tuplas = zip(sol_1, sol_2)
  
  for tupla_1, tupla_2 in tuplas:
    if tupla_1 < tupla_2:
        domina = True
    
    return domina 

def fronteira_pareto_simples(solucao):
    nova_lista = []
    for sol in solucao:
      nova_lista.append(sol)
      
      for sol_aux in nova_lista:
        if sol_aux != sol:
          if dominancia_simples(sol_aux.custo, sol.custo):
            nova_lista.remove(sol)
            break
          elif dominancia_simples(sol.custo, sol_aux.custo):
            nova_lista.remove(sol_aux)
    return nova_lista[0]

#Tweak proposto
def algorithm_eight(solucao, d, p, r):
    actual_sol = []
    for i in range(d):
        actual_sol.append(solucao[i])
        
    for i in range(d):
        prob = random.randrange(0, 1)
        if(prob < p):
            valor = actual_sol[i] + random.randrange(-1 * r, r)
            while (valor > 100 or valor < -100):
                valor = actual_sol[i] + random.randrange(-1 * r, r)
            actual_sol[i] = valor
            
    return actual_sol

#Crossover proposto
def algorithm_24(vetorA, vetorB, d):
    e = random.randrange(0, d)
    f = random.randrange(e, d)
    
    vetor_aux = []
    for i in range(d):
        if i <= e:
            vetor_aux.append(vetorA[i])
        if i > e and i <= f:
            vetor_aux.append(vetorB[i])
        if i > f:
            vetor_aux.append(vetorA[i])
    return vetor_aux
